## Rust Crypto

Cryptographic algorithms written in pure Rust.

### About

We maintain hundreds of crates which provide pure Rust implementations of cryptographic algorithms,
including the following:

- `aead`
- `aes-gcm`
- `blake2`
- `chacha20poly1305`
- `cipher`
- `const-oid`
- `crypto-bigint`
- `der`
- `digest`
- `ecdsa`
- `ed25519`
- `hmac`
- `keccak`
- `pbkdf2`
- `pkcs8`
- `rsa`
- `sha2`
- `sha3`
- `signature`
- `zeroize`
